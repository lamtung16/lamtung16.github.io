II. Life Is Faster ‚Äî Compression of Experience

Speed alters form. Longer practices compress into briefer versions, changing rituals and memory formation.

Long movies
Short-episode TV
Full, shared meals
Fast food

Effect: long-term, coherent memories can be replaced by many short, disconnected impressions.

III. Analogies ‚Äî Mental Models to Remember By

Use these mental images to talk about the trade-offs in conversation.

Muscle & Exoskeleton

Tools are exoskeletons: they boost power immediately but reduce the need to build raw muscle over time.

River & Pipe

Skills carve deep paths like rivers; tools act like pipes that move water quickly but leave no landscape behind.

Map & GPS

Using your memory to navigate is drawing your own map. GPS is a chauffeured ride ‚Äî you arrive but you no longer own the route.

Fireplace & Microwave

Tending a fire teaches process and patience; microwaves teach only instant convenience.

Library & Search Engine

A library visit builds a mental neighborhood; a search engine teleports you to an answer and leaves fewer memory traces.

Tool as Prosthesis

Tools can be prostheses that extend capability ‚Äî but if we never exercise the original limb, it atrophies.
